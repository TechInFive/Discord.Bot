# Discord Bot 🐍

## Features 🎮

- Discord Bot

## Prerequisites 🛠

- Python 3.x
- Discord library

## Setup & Run 🚀

1. Clone the repository:
```
git clone https://github.com/TechInFive/Discord.Bot
cd Discord.Bot
```

2. Install the required packages:
```
pip install discord.py
```
